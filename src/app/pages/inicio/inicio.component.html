Projeto Angular de Gestão de Produtos - Avaliação Minsait

Descrição do Projeto

Este projeto consiste em uma aplicação Angular para gerenciar produtos, sendo uma avaliação do curso de capacitação da Minsait. A aplicação permite listar, cadastrar, editar e deletar produtos de uma loja.

Funcionalidades

  • Listagem de Produtos: Exibe uma tabela com os produtos, incluindo id, nome, código de barras, preço e opções de editar/deletar.
  • Cadastro de Produtos: Permite adicionar novos produtos com nome, código de barras e preço, com validações nos campos.
  • Edição de Produtos: Possibilita a edição de produtos existentes, mantendo a mesma estrutura de campos e validações.

Arquitetura do Projeto

O projeto segue uma arquitetura de componentes Angular para separar as funcionalidades em diferentes partes do aplicativo. Os componentes principais incluem:

  • produtos.component: Para exibir a lista de produtos e deletar produtos existentes.
  • cadastra-produtos.component: Para adicionar novos produtos.
  • editar-produtos.component: Para editar produtos existentes.

Comunicação com a API Spring

Para buscar, cadastrar, editar e excluir produtos, a aplicação se comunica com uma API Spring desenvolvida na aula de Spring do curso de capacitação. Os serviços HTTP são usados para interagir com a API e os dados dos produtos.
